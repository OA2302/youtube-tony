Text color

In HTML you can change the color of text using this syntax:
< h3 style="color:blue"> This text is blue < /h3 >
output

This text is blue


< h3 style="color:red"> This text is red < /h3 >
output

This text is red


< h3 style="color:white"> This text is white < /h3 >
output

This text is white


background color

To add a background-color to your website just simply use the Syntax:
< h3 style="background-color: cadetblue;" > I am in a cadetblue background < /h3 >

output

I am in a cadetblue background

< h3 style="background-color: orange;" > I am in a orange background < /h3 >

output

I am in an orange background

< h3 style="background-color: purple;" > I am in a purple background < /h3 >

output

I am in a purple background

Note:To apply a whole body background-color. simply use
< body style="background-color: blue;" >< /body >
